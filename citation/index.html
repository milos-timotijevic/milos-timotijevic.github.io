How to Cite the Website and Data

Ова страница обједињује препоручени начин цитирања академског сајта, библиографије, графа знања и отворених података Милоша Тимотијевића.

This page provides citation guidance for the academic website, bibliography, knowledge graph and open data of Miloš Timotijević.

Препоручени цитат / Recommended Citation

Тимотијевић, Милош. Милош Тимотијевић — академски профил и граф знања. Континуирано ажуриран академски сајт и отворени библиографски скуп података. https://milos-timotijevic.github.io/. Приступљено: [унесите датум].

Timotijević, Miloš. Miloš Timotijević — Academic Profile and Knowledge Graph. Continuously updated academic website and open bibliographic dataset. https://milos-timotijevic.github.io/. Accessed [insert date].

Машински читљиви подаци / Machine-Readable Metadata

Канонски подаци за цитирање доступни су у формату Citation File Format. GitHub их користи за приказ цитата у APA и BibTeX облику.

Canonical citation metadata are available in Citation File Format. GitHub uses them to generate APA and BibTeX citations.

За цитирање појединачне публикације треба користити библиографске податке, DOI, ISBN, COBISS или други идентификатор наведен уз ту публикацију.

To cite an individual publication, use the bibliographic metadata and the DOI, ISBN, COBISS or other identifier given with that publication.
